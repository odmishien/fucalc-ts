# Usage
<!-- usage -->
```sh-session
🐑🌙 npm install -g fucalc-ts
🐑🌙 fucalc-ts
? 1番目のメンツはどれですか？ 中張牌|暗刻子(🀉 🀉 🀉 )
? 2番目のメンツはどれですか？ ヤオ九牌|暗刻子(🀇 🀇 🀇 )
? 3番目のメンツはどれですか？ 明順子(チー:🀈 🀉 🀊 )
? 4番目のメンツはどれですか？ 暗順子(🀈 🀉 🀊 )
? アタマはどれですか？ 客風牌
? 待ちの形はどれですか？ 両面
? アガリ方はどれですか？ ツモ

🀄️あなたの手は 40 (合計:34) 符です🀄️

🐑🌙 fucalc-ts (-v|--version|version)
fucalc-ts/1.0.3 darwin-x64 node-v14.4.0
```
