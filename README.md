<!-- usage -->
```sh-session
$ npm install -g fucalc-ts

? 1番目のメンツはどれですか？ 暗順子(🀈 🀉 🀊 )
? 2番目のメンツはどれですか？ 暗順子(🀈 🀉 🀊 )
? 3番目のメンツはどれですか？ 暗順子(🀈 🀉 🀊 )
? 4番目のメンツはどれですか？ 暗順子(🀈 🀉 🀊 )
? アタマはどれですか？ 場風牌
? 待ちの形はどれですか？ 単騎
? アガリ方はどれですか？ ツモ

🀄️あなたの手は 40 (合計:36) 符です🀄️
```
